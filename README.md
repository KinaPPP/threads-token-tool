# 🧵 Threads API トークン取得支援ツール

このツールは、ブラウザ拡張機能や外部アプリで Threads API を利用するために必要な **「長期 Access Token（60日間有効）」** と **「User ID」** を、ブラウザ上の操作だけで簡単に取得するためのヘルパーページです。

🌐 **[トークン取得ツールを開く](https://[あなたのユーザー名].github.io/threads-token-tool/)**
*(※利用する際は、上記のリンクをクリックしてください)*

---

## 🔒 セキュリティとプライバシーについて

このツールは**完全にあなたのブラウザ内（ローカル環境）でのみ動作**します。
バックエンドサーバーは存在せず、入力された「App ID」「App Secret」や、取得した「Access Token」などの機密情報が、**外部のサーバーや第三者に送信・保存されることは一切ありません。**

取得処理が終わってページを閉じた瞬間に、すべてのデータはブラウザから完全に消去されます。安心してご利用ください。

---

## 📝 使い方

### 事前準備：Meta for Developers でのアプリ設定
このツールを使う前に、ご自身の Meta 開発者アカウントでアプリを作成し、設定を行う必要があります。

1. [Meta for Developers](https://developers.facebook.com/) にログインし、「マイアプリ」から新しいアプリを作成します。
2. ユースケース等を選択し、アプリに **「Threads API」** を追加します。
3. Threads API の設定画面（左メニューの「Threads API」＞「設定」）を開き、以下の情報を確認・設定します。
   - **ThreadsアプリID (App ID)** をメモする。
   - **Threadsのapp secret (App Secret)** をメモする。
   - **「有効なOAuthリダイレクトURI」** 等の必須入力欄に、**このツールのURL（ `https://[あなたのユーザー名].github.io/threads-token-tool/` ）をコピペして、Enterキーを押して確定**させ、「変更を保存」します。

### トークンの取得手順
1. 本ツールのページを開きます。
2. 事前準備でメモした **App ID** と **App Secret** を入力し、「Threadsと連携してトークンを取得する」ボタンを押します。
3. Threads（Instagram）のログイン画面・連携許可画面が表示されたら、内容を確認して許可します。
4. 元の画面に戻ると、緑色の枠で **長期 Access Token** と **User ID** が表示されます。
5. 「コピー」ボタンを押して、お使いの拡張機能（またはアプリ）の設定画面に貼り付けてください。

## 🛠 技術的な仕組み
* **完全静的ページ**: HTML / CSS / Vanilla JavaScript のみで構成されており、GitHub Pages 等でホスティング可能です。
* **クライアントサイドOAuth**: `fetch` API を使用して、Meta の Graph API と直接通信してトークンを交換します。
* **セッションの即時破棄**: リダイレクト時の状態保持に使用した `sessionStorage` は、トークン取得処理の開始と同時に即座に破棄されます。
